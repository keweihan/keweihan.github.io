Warpball Development Log

June 15th 2021

Changes: 
- Added square obstacles
    - Different triggers for each face. Contact with face facing player will send to menu 
- Added basic menu
- Added basic manual boost system and UI 
    - Boost with spacebar
    - Cannot boost if no boost charge
    - Get boost charge by collecting yellow cubes
    - Boost charge displayed at bottom of screen
- Obstacles now despawn when player has passed them'
- Obstacles now spawn in round numbers (adjustable - i.e. every 5 units)
- Added basic GameManager (future TODO - move variables to be stored here)
- Changed strafe movement to be velocity based instead of using transform
    - this removes glitches/inaccuracies with strafing into hitboxes (i.e. used to slightly 
    clip into obstacle and trigger front face)
- Floor is now infinite
