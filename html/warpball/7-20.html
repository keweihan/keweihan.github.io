Warpball Development Log

July 20th 2021

Changes: 
- Added out of boost indicator flashes (not shown)
- Added ability to strafe boost with keys Q/E
- Added basic UI to display score, current level, score registration on ramp
    - Animated with 3rd party library LeanTween
- Added scoring system and difficulty/level scaling with time
    - Uses a general GameManager script
    - Collectible scoring 
        - Score value of collectible increases with each collect
            - introduces reward in risk/reward of chaining multiple collectibles
        - Collecting collectible increases player speed until ramp
            - introduces risk in risk/reward of chaining multiple collectibles
        - Value and speed increases from collectible resets on ramp
        - Score collected from collectibles registered on ramp
    - Level/Difficulty scaling
        - Speed permenantly increases with each level 
        - Number of obstacles generated permenantly increases with each level
        - Level increases by 1 every 15 seconds (adjustable)
        - incentivizes risking for reward (before levels get inevitably too difficult)
        - from Tetris 
- Added basic player model (temporary?) with script based animations
    - With particle effects
    - Tilting animation on moving left/right
    - Tilting up/down animation based on movement direction during flight
    - Hovering up/down animation  
    - Rolling animation on strafe boost
- Minor visual tweaks
    - floor color
    - post processing effects (bloom, distortion)
