Warpball

Warpball is a personal coding project. The goal is to create a simple arcade game based on an old mobile arcade infinite runner "Warp Rush", with a few original ideas. Progress is logged in posts below.

Q: Why the 'ball' in 'warpball'?

A: In theme with being a learning project, this is a juiced up version of Roll-a-ball. The underlying player is actually still the ball from the Roll-a-ball tutorial.
